# UK Rental Price Prediction System

This project presents a machine learning-based system for predicting rental property prices in the UK housing market. The system uses property features such as city location, number of rooms, and property type to estimate the monthly rental price of a property.

The application was developed as part of the **CIS4055 Computing Masters Project at Teesside University**.

---

## Project Overview

The aim of this project is to design and evaluate a machine learning pipeline capable of predicting rental prices using housing data. The system includes:

- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Machine learning model training and evaluation
- Model comparison
- Feature importance analysis
- Deployment of an interactive prediction system using Streamlit

The final system allows users to input property details and obtain an estimated rental price generated by the trained machine learning model.

---

## Dataset

The dataset used in this project was obtained from Kaggle.

Dataset Source:

https://www.kaggle.com/datasets/georgejnr/uk-rental-properties-dataset

The dataset contains rental property listings including:

- Property description
- Location
- Number of rooms
- Rental price

After preprocessing and removing duplicate records, the final dataset used for modelling contained **1,993 property listings**.

---

## Machine Learning Models

The following machine learning algorithms were implemented and evaluated:

- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor
- XGBoost Regressor

Model performance was evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

The best-performing model was selected and deployed in the prediction system.

## Web Application

The trained machine learning model was deployed as an interactive web application using **Streamlit**.

The application allows users to input:

- City
- Number of rooms
- Property type

The system then generates an estimated monthly rental price based on the trained model.

The application will open in your browser.
